# README

### Rails Initialized
* `rails new -T -C -P --skip-turbolinks --database=postgresql acklen`
* `rails generate rspec:install`

### Gitpod
* https://gitpod.io/#prebuild/https://github.com/udit7590/acklen
* https://amber-locust-mtoalhgr.ws-us11.gitpod.io/

### Configure
* `bundle install`
* `yarn install`
* `rails db:create`
* `rails db:migrate`
* `rails db:seed`

### Start Server
* `rails s`
